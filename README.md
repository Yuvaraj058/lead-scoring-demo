# AI Lead Intelligence Demo – Euprime

## Objective
Identify, enrich, and rank high-probability leads interested in 3D in-vitro models for therapy development.

## What This Demo Shows
- Business-driven scoring logic (0–100)
- Scientific + funding + role-based prioritization
- Searchable and exportable dashboard

## Why Sample Data?
Live LinkedIn and Crunchbase data require paid APIs and legal access.
This demo focuses on system logic and decision intelligence.

## Tech Stack
- Python
- Pandas
- Streamlit

## How It Scales
- LinkedIn → Proxycurl API
- Publications → PubMed API
- Funding → Crunchbase / PitchBook
- ML-based ranking in next iteration
